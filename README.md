## 🧩 Tecnologias e Serviços Utilizados

- AWS EC2 (Instâncias Ubuntu)

- MongoDB
- VPN (OpenVPN ou similar)
- HTML/CSS simples (para teste de backend)
- SSH para provisionamento manual
